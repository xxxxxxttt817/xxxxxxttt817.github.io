Collection小结

Veröffentlicht am 2016-10-28

Collection特点：
1.用于存储对象的容器
2.长度可变
3.不可以存储基本数据类型值 存储基本数据类型，用包装类

框架顶层Collection常见方法：
1.添加：
boolean add
boolean addAll(Colloction coll)
2.删除：
boolean remove(object obj)
boolean removeAll(Colloction coll)
boolean clear(Colloction coll)
3.获取：
int size() //获取集合中元素的个数
Iterator Iterator()
该对象必须依赖于具体容器，因为每一个容器的数据结构都不同。
所以该迭代器对象是在容器中进行内部实现的。
对于使用容器者而言，具体的实现不重要，只要通过容器获取到该实现额迭代器的对象即可。
也就是iterator方法。

4.判断
boolean isEmpty()
boolean contains(object obj)
boolean contains(object obj)

迭代器：
class Demo{
public static void main(String[] args){
Collection coll = new ArrayList();

    coll.add("1");
    coll.add("2");
    coll.add("3");
    coll.add("4");

/*
    两种方法使用Iterator取出Collection中的元素
*/    

    //方法1：使用了Collection中的iterator()方法。
    Iterator i = coll.iterator();
    if (i.hasNext()){
        i.next();
    }

    //方法2：
    for(Iterator it = coll.iterator(); it.hasNext();){
        System.out.println(it.next());
    }
}

}
